## 2. 반복문

특정 조건을 만족할 동안 반복하라는 문법

### ✔ for문
```js
for (초기화식; 조건식; 증감식){
    // 조건식이 참인 경우 반복 실행될 코드;
}

for (var i=0; i<10; i++){
    console.log(i);
}
```
### ✔ while문
```js
while (조건) {
    // 조건이 참일 동안 반복할 코드
}
```
