# 프레임워크를 위한 JavaScript ES6

JavaScript는 **'객체'** 기반의 **'스크립트 프로그래밍 언어'**

### ➰ Intro.
- 웹을 이루는 가장 중요한 세가지 요소

    : 구조 스타일 논리(JS)

- JavaScript의 역할

    : 상황에 맞게 사용자와의 상호작용에 따라서 html/CSS를 달리 보여주는 역할

- 크로스 브라우징 이슈

    : js는 브라우저 상에서 동작하는 언어 ->  다양한 브라우저가 존재 -> 표준이 필요!

    -> ECMAScript

- ECMAScript (ES6)
